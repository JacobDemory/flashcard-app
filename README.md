# Flashcard App

A browser-based study tool for creating, reviewing, and organizing flashcards. Add custom cards, filter by category, track progress, and save your deck locally in the browser.

---

## Features
Recent additions are in **bold**
- Click a flashcard to flip between the term and definition
- Navigate through cards with Previous and Next buttons
- Add custom flashcards with terms, definitions, and categories
- **Save flashcards locally using browser localStorage**
- **Filter cards by category**
- **Shuffle the current study deck**
- **Track total cards, reviewed cards, mastered cards, and progress percentage**
- **Mark cards as mastered while studying**
- **Delete cards from the current deck**
- Responsive layout for desktop and mobile screens

## How to Run
1. **Prerequisites**: A modern web browser such as Chrome, Firefox, Safari, or Edge.
2. **Download**: Clone or download this repository to your local machine.
3. **Run the App**: Open `index.html` in your browser.

---

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JacobDemory/flashcard-app.git
   ```
2. Navigate to the directory:
   ```bash
   cd flashcard-app
   ```
3. Open `index.html` in your browser.

---

## Example Usage
1. Select a category or choose all cards.
2. Click the flashcard to flip between the term and definition.
3. Use Previous and Next to move through the deck.
4. Add new cards using the form.
5. Mark cards as mastered to track study progress.
